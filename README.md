## 核心创新: 表征感知课程学习 (erankOnline)

传统课程学习只看 loss。我们引入**表征维度**: 每 500 epoch 提取 penultimate 层 128 维激活, 计算每个样本的局部有效秩 (erank)。

```
难度 = loss_score × (1 - α) + (1 - erank_score) × α
α 从 0 逐渐增长到 0.5
```

erank 高 = 基底丰富 = 真简单; erank 低 = 基底塌缩 = 真难。相比标准训练: **买入 -57%, 年均 +1.5pp, 熊年首次翻正**。

## 关键发现

1. **买入价定义决定一切**: 季度截止日买入存在前视偏误 (年化虚高), 固定延迟后回归真实
2. **erank > 纯 loss**: 表征维度的有效秩比纯 loss 更能区分真/假难样本, 熊年表现明显提升
3. **行业轮动自动化**: 模型在不同年份自动偏好不同行业, 无需手动调仓
4. **诚实基线让数字可信**: 等权全买 +12.3%, 模型 +21.0%, 超额 8.7pp
