## 核心创新：表征感知课程学习

传统课程学习只看 loss——样本 loss 小就是简单。我们引入**表征维度**：每 500 epoch 提取网络倒数第二层（128 维）的激活向量，计算每个样本的局部有效秩（erank）。

```
难度 = loss_score + (1 - erank_score) × α
```

初期 α=0（纯 loss 主导），后期 α→0.5（erank 逐渐介入）。erank 高 → 基底丰富 → 真简单；erank 低 → 基底塌缩 → 真难。

相比标准训练：**买入 -57%，年均 +1.5pp，熊年首次翻正**。

## 关键发现

1. **买入价定义决定一切**：季度截止日买入存在前视偏误（年化虚高 30pp），固定延迟后回归真实
2. **erank > 纯 loss**：表征维度的有效秩比纯 loss 更能区分真难和假难样本，熊年表现明显提升
3. **行业轮动自动化**：模型在不做任何手动调整的情况下，在不同年份自动偏好不同行业
4. **诚实基线让数字可信**：等权全买 12.3%，模型 21.0%，超额 8.7pp，每个数字都可复现
